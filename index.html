Book Search Application - IT 4403

Overview

This project is a single-page application (SPA) designed to search and display book information using the Google Books API. You can search for books, view details of individual books, and organize book lists.

Project Details

Milestone Description
Milestone 1
  • Single Book Display - Display a single book's details from a JSON file.
  • Book List - Display a list of books from a JSON file.
Milestone 2 API Integration
Milestone 3 AJAX/AJAJ Operations
Milestone 4 SPA UI Enhancements
